# Clínica Veterinária
O sistema deve permitir gerenciar as consultas realizadas na clínica, deve registrar
os dados do cliente como CPF, nome, os dados dos animais como nome, raça, idade e descrição
das características do animal. Deve permitir registrar os dados dos exames e remédios gerados 
nas consultas bem como a quantidade de exames ou medicamentos fornecidos, data do atendimento e
anotações geradas pelo veterinário, valores dos medicamentos e exames. Também deve permitir 
verificar quais veterinários atenderam a quais consultas, sobre o veterinário armazenar CFMV, nome e especialidade
